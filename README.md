# MedTrack V1
Hospital Equipment Maintenance Management System.

## Features
- Dashboard with Equipment KPIs
- Automated Maintenance Scheduling
- Repair Tracking & History
- Multi-user Authentication (Admin/Technician)
- Financial Spending Analysis (Chart.js)
